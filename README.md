# This is a Wi-Fi remote-controlled boat based on the ESP8266, which has been proven in practice.
# 这是一艘基于8266的wifi控制小船

是我和我的朋友在闲时创作的，难度不大，但我希望所有人可以快速地找到一个项目创建的方向
It was created by my friend and me in our spare time. It wasn't very difficult, but I hope it can help everyone quickly find a direction for creating a project.

接下来我会介绍如果你需要快速复刻这个项目的步骤
Next, I will introduce the steps if you need to quickly replicate this project.

## 1：下载model文件夹中的模型文件，在写这篇readme时我没有收集到项目过程中的所有模型文件，后续会陆续上传
## 1. Download the model files in the model folder. At the time of writing this readme, I haven't collected all the model files from the project; I will upload them gradually later.

## 2：现在你应该拥有了船底的模型，把他使用3d打印机或者别的方法变为实物
## 2: Now you should have a model of the ship's hull. Use a 3D printer or other methods to turn it into a physical object.

## 3：下载pcb文件夹中的pcb gerber文件，在制作pcb之前最好先验证一下其可行性
## 3. Download the PCB Gerber file from the PCB folder. It's best to verify its feasibility before creating the PCB.

## 4：根据bom表购买所需的元器件，然后焊接好
## 4. Purchase the required components according to the BOM (Bill of Materials) and then solder them.

## 5：除了bom表外，我们使用了一个n30电机作为船的推进装置，你还需要一个mg90s舵机以及一个电池
## 5: In addition to the BOM (Bill of Materials) gauge, we used an N30 motor as the boat's propulsion system. You will also need an MG90S servo motor and a battery.

## 6：现在你可以把pcb，电机，舵机都装进船舱中
## 6: Now you can install the PCB, motors, and servo into the cabin.

## 7：如果一切顺利，你应该可以搜索到esp8266的wifi信号，连接他并且在浏览器中打开此网址 192.168.4.1
## 7. If everything goes smoothly, you should be able to find the ESP8266's Wi-Fi signal. Connect to it and open this URL in your browser: 192.168.4.1

## 8：现在，愉快地游玩！
## Now, have fun!
